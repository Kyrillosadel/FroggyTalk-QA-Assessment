# FroggyTalk-QA-Assessment
Setup:
- Download and Install Python 3.x 
- After installing python open command prompt to Install dependencies:
      pip install pytest requests Appium-Python-Client
- Download tests from the Repo and Navigate to the directory in which your API Tests exist then Run:
      pytest test_api.py
- For Running Mobile Tests:
- Start Appium server:
      appium
- Run:
      pytest test_ui.py
